Le plugin "TabNine"

Ce plugin permet d'ajouter Une auto-complétion à sublime text. Cette auto-complétion est basée sur un modèle de deep learning, et elle est donc beaucoup plus efficace qu'une auto-complétion classique.

Ici pour sa page sur packagecontrol.io

Quel avantages

L'auto-complétion TabNine est avantageuse pour plusieurs raisons

  • Elle est basé sur un modèle de deep learning. Il est donc très efficace pour le code (il détecte les structures que vous avez déjà écrites)

  • Elle peut s'intégrer à l'auto-complétion de base, mais aussi former une deuxième auto-complétion, ce qui permet d'avoir deux résultats en même temps : l'un avec la touche 'Tab', et l'autre avec la touche 'Entrée'. Le désavantage est que TabNine désactive l'autocomplétion de sublime text à chaque démarrage de l'application...
