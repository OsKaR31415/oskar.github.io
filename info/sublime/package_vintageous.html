Le package "Vintageous"

Ce package permet d'utiliser les raccourcis Vim sur sublime text. Le package Vintage, qui est installé par défaut sur sublime text, fait la même chose, mais Vintageous à l'avantage d'être plus maintenu et plus complet

Que sont les raccourcis Vim

Les raccourcis clavier de l'application Vim (une application qui fonctionne dans les terminaux Unix)
