Sublime Text

Sublime Text est un éditeur de code extensible. Il à l'avantage d'être à la fois léger et puissant, grâce au système des packges.

plugins

Sublime Text est très extensible : de nombreux plugins sont créés afin d'ajouter des fonctionnalités très variées, et pour tous les langages. Les plugins de sublime text sont programmées en python, mais ils sont malgré tout très rapides.

Voici une liste des plugins que je trouve les plus utiles :

vintageous : Les raccourcis clavier Vim

Bracket Highlighter : pour montrer structures de parenthèses, tags etc... dans le code

LiveCoding in python : Pour visuialiser en temps réel, ligne par ligne, l'execution d'un code python (très utile pour apprendre le python)

TabNine : Une auto-complétion très efficace !

VintageLines : Un plugin qui va bien avec Vintageous (permet d'avoir une numérotation relative)
