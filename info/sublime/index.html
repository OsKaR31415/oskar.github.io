Sublime Text

Sublime Text est un éditeur de code extensible. Il à l'avantage d'être à la fois léger et puissant, et malgré tout très extensible.

Packages

Sublime Text est très extensible : de nombreux packages sont créés afin d'ajouter des fonctionnalités très variées, et pour tous les langages. Les packages de sublime text sont programmées en python, mais ils sont malgré tout très rapides.
