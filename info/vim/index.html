Vim

Vim est un éditeur de texte spécialisé dans le code.

Cet éditeur est merveilleux car il utilise des raccourcis clavier extrêmement puissants. Vim est en fait basé sur l'idée que l'on modifie le code plus souvent que l'on écrit du code nouveau (ce qui est quasiment toujours vrai).

L'édition modulaire

Vim utilise ce qu'on appelle de l'édition modulaire (comme l'éditeur emacs par exemple), c'est-à-dire qu'il existe plusieurs modes différents, et qu'une même touche ne fait pas du tout la même chose dans deux modes différents.

Le mode principal de Vim est le mode "normal" (normal mode en anglais). Dans ce mode, on ne peut pas taper de texte : chaque touche du clavier sert pour un raccourci. Pour taper du texte, il faut entrer en mode "insertion" (insert mode). Il existe également d'autres modes, comme le mode "visuel" (visual mode) pour sélectionner du texte, ou le mode "commande" (command mode) pour taper des commandes sur le fichier (saufegarder, quitter...) ou bien sur des aspects visuels (gérer les onglets, les fenêtres etc...) ainsi que les réglages.
